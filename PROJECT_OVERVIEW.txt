📞 故障排查
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  问题: Neo4j 启动失败
  解决: 检查端口 7474, 7687 是否被占用
       netstat -ano | findstr "7474"

  问题: API 服务启动失败
  解决: 确认依赖已安装
       pip install -r requirements.txt

  问题: 查询结果为空
  解决: 确认数据已成功导入
       访问 http://localhost:7474 验证

  更多问题请参考 README.md 的"常见问题"部分
